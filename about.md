# MM Serp Crawler

Google Chrome extension untuk get/grab url dari hasil pencarian Google Search Engine Result Page (SERP) berdasarkan nama domain yang sudah di add.

Ini adalah extension Google Chrome yang fitur dan fungsinya:

# fitur

- ada opsi untuk add dan memasukan nama domain misal: codef.id
- ada tabel berisi nama domain yang sudah di add
- ada tombol untuk menghapus nama domain yang sudah di add
- ada tombol untuk mengedit nama domain yang sudah di add
- ada tombol untuk menyimpan perubahan nama domain yang sudah di edit
- ada tombol untuk membatalkan perubahan nama domain yang sudah di edit
- ada tombol untuk menghapus semua nama domain yang sudah di add

# fungsi

- saya tekan icon extension di toolbar Google Chrome
- tampil popup extension yang berisi tombol untuk menjalankan get/grab url dari hasil pencarian Google.
- otomatis urls akan ter-copy ke clipboard
